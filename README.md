# Employee-Management-System


Overview
A Full Stack Employee Management System developed using Spring Boot, React.js, and MySQL. The application demonstrates full-stack development by providing a responsive user interface and RESTful APIs to perform complete employee management operations.

Features
- Employee Registration
- View All Employees
- Update Employee Information
- Delete Employee Records
- Search Employee Details
- Responsive User Interface
- REST API Integration
- MySQL Database Connectivity

Tech Stack
Frontend : React.js,  JavaScript (ES6), HTML5, CSS3, Axios
Backend : Java, Spring Boot, Spring MVC, Spring Data JPA, Hibernate, REST APIs
Database : MySQL
Tools : Maven, Git, GitHub, Postman, IntelliJ IDEA, VS Code

Concepts Demonstrated
Backend
- RESTful Web Services
- Spring Boot Framework
- Spring MVC Architecture
- Spring Data JPA
- Hibernate ORM
- CRUD Operations
- Exception Handling
- Layered Architecture
Frontend
- React Components
- React Hooks
- State Management
- API Integration using Axios
- Form Validation
- Responsive UI Design
 Database
- Relational Database Design
- MySQL
- Entity Mapping using JPA

REST API Endpoints
| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/employees` | Get All Employees |
| GET | `/api/employees/{id}` | Get Employee By ID |
| POST | `/api/employees` | Add New Employee |
| PUT | `/api/employees/{id}` | Update Employee |
| DELETE | `/api/employees/{id}` | Delete Employee |
